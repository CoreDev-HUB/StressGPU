## 🛠️ Overview
**StressGPU** is a high-performance benchmarking tool that uses **Raymarching** and **Fractal Mathematics** to push your graphics card to its limits. Unlike standard benchmarks, it renders a dynamic Mandelbulb fractal in real-time, forcing the GPU to execute complex trigonometric kernels at high clock speeds.

## ✨ Features
* **Extreme Stress:** 8-iteration fractal kernels for maximum thermal load.
* **RGB Dynamic Shading:** Real-time color flow based on spatial coordinates and time.
* **Live Kernel Injection:** Modify the GPU math on-the-fly via the "Kernel Tweak" console.
* **Zero Installation:** Runs directly in the browser using raw WebGL.

## ⚠️ Disclaimer
This tool is designed to generate significant heat. Use at your own risk. Ensure your cooling system (especially on laptops) is working correctly before running extreme kernels.
